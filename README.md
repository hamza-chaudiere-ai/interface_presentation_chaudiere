# 🔥 Interface Présentation Chaudière

Ce projet propose une interface web complète pour visualiser les composants d’une chaudière industrielle, comprendre les causes de percement, explorer les types de contrôles réalisés, et afficher une AMDEC dynamique générée à partir d’un fichier Excel.

---

## 📌 Fonctionnalités

- 📁 Navigation par composants (images + définitions)
- 🧠 Causes de percement (texte explicatif + visuel)
- 🛠️ Contrôles réalisés (illustrés + expliqués)
- 📊 AMDEC importée depuis fichier Excel avec mise en couleur automatique
- 📈 Dashboard Power BI : températures entrée/sortie de la chaudière

---

## ⚙️ Technologies

- HTML, CSS, JavaScript
- Python pour la génération AMDEC (`amdec_generator.py`)
- Power BI (Dashboard)
- GitHub Pages (hébergement)
